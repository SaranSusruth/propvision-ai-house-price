# 🏛️ PropVision AI — House Price Intelligence Platform

## 🚀 Live Demo  
👉 https://propvision-ai-house-price.streamlit.app/

An AI-powered real estate valuation platform that predicts house prices using machine learning, enhanced with demand analysis, zone-based market intelligence, and future price projections.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🤖 ML Valuation | Random Forest model trained on multi-feature property data |
| 🔥 Dynamic Demand Engine | Real-time demand score based on zone, location quality & area size |
| 📈 Inflation Projections | 10-year price trajectory with zone-specific appreciation rates |
| 🏘️ Zone Comparison | Side-by-side Rural / Suburban / Urban / Luxury market analysis |
| 🔬 Feature Importance | Visual explainability of what drives property prices |
| 💎 Luxury UI | Dark theme, animated cards, Plotly charts, professional grade design |

---

## 🚀 Quick Start

### Step 1 — Install dependencies
```bash
pip install -r requirements.txt
```

### Step 2 — Train the model
```bash
python train_model.py
```
This generates `model.pkl` in the same folder.

### Step 3 — Run the app
```bash
streamlit run app.py
```

The app opens at **http://localhost:8501**

## 🧠 How the Demand Engine Works

```
Demand Score = 0.55 × Zone Base + 0.30 × Location Quality + 0.15 × Area Size

Zone Demand Bases:
  Luxury   → 88%    Inflation: 8.5%/yr
  Urban    → 75%    Inflation: 7.2%/yr
  Suburban → 58%    Inflation: 5.5%/yr
  Rural    → 35%    Inflation: 3.2%/yr

Demand Multiplier = 1.0 + Demand Score × 0.35
Final Price = AI Base Price × Demand Multiplier
```
